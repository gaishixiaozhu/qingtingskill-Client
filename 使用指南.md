# 蜻蜓志愿数据查询 Skill — 内部测试使用指南

## 第一步：安装 Skill

### 方式一：从 GitHub 安装（推荐）

```bash
# 在 OpenClaw 工作目录下执行
cd ~/.openclaw/workspace/skills

# 克隆 skill
git clone https://github.com/gaishixiaozhu/qingtingskill-Client.git qingting-query

# 安装依赖
cd qingting-query
pip3 install requests
```

### 方式二：手动安装

把 `qingtingskill-Client` 文件夹放到 `~/.openclaw/workspace/skills/` 下即可。

`config.py` 中的服务器地址已配置为：
```python
SERVER_URL = "https://ocs.qingtingai.pro"
```

无需修改。

## 第二步：配置 API Key

在终端设置环境变量（或写入 `~/.zshrc` / `~/.bashrc`）：

```bash
export QINGTING_API_KEY="你的Key"
```

## 第三步：测试

```bash
cd ~/.openclaw/workspace/skills/qingting-query

# 健康检查
python3 qingting_query_skill.py --api-key $QINGTING_API_KEY

# 测试查询
python3 qingting_query_skill.py -q "山东物理600分推荐计算机专业" --api-key $QINGTING_API_KEY
```

## 第四步：在 OpenClaw 中使用

安装完成后，直接跟小助说类似的话即可触发：

> "辽宁物理类520分推荐什么学校"
> "浙江600分能上什么大学"
> "广东历史类550分计算机专业"
> "四川2025年一本线多少"

## 支持的查询类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 志愿推荐 | 根据分数推荐冲/稳/保 | "辽宁物理520分推荐计算机" |
| 招生计划 | 查招生人数 | "大连理工在山东招多少人" |
| 历史分数 | 查历年录取分 | "华中科技计算机去年多少分" |
| 一分一段 | 分数↔位次互查 | "浙江600分排多少名" |
| 批次线 | 省控线查询 | "四川2025年一本线多少" |
| 院校信息 | 查学校背景 | "深圳大学是双一流吗" |
