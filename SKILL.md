# 蜻蜓志愿数据查询 Skill v2.0

## 功能

蜻蜓生涯志愿填报数据库全功能查询服务。只要是志愿填报相关数据查询，都通过此 Skill 处理。

## 支持的查询类型

| 查询类型 | 能做什么 | 示例问题 |
|---------|---------|---------|
| **志愿推荐** | 根据分数/位次推荐院校专业，按冲刺/适合/稳妥三层分类 | "辽宁物理类520分推荐什么学校" |
| **招生计划** | 查某校某专业在某省的招生人数 | "大连理工在辽宁招多少人" |
| **历史分数** | 查院校专业历年录取分数线 | "华中科技计算机去年多少分" |
| **一分一段表** | 分数↔位次互查 | "辽宁物理类600分排多少名" |
| **批次线** | 查各省各批次省控线 | "辽宁2025年一本线多少" |
| **院校信息** | 查院校代码、层次、所在地等 | "大连理工是985吗" |
| **专业查询** | 查专业介绍、学什么、就业方向 | "计算机科学与技术学什么" |

## 配置

### 环境变量

```bash
export QINGTING_API_KEY="tk_你的APIKey"
```

### config.py

```python
SERVER_URL = "https://ocs.qingtingai.pro"
```

## 使用方法

用户发送问题 → Skill 自动转发到服务器 → 返回结构化答案。

### 志愿推荐时的三必填条件

生成志愿方案必须确认三项信息，缺一不可：
- ① **考生省份**（如辽宁、山东）
- ② **选科**（物理类/历史类，或文科/理科）
- ③ **分数或位次**

缺少时自动追问补全。

## API Key

联系管理员获取。每个 Key 有独立的日配额和月配额。
