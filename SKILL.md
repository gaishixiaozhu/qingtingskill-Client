# 蜻蜓志愿数据查询 Skill

## 功能

通过此Skill，客户端可以向Server OpenClaw发送志愿填报相关问题，获取答案。

## 配置

### 首次使用

在Skill目录的 `config.py` 中配置：

```python
# Server地址
SERVER_URL = "http://114.215.127.115:5007"

# API Key（联系管理员获取）
# 在环境变量 QINGTING_API_KEY 中设置
```

或者设置环境变量：
```bash
export QINGTING_API_KEY="tk_你的APIKey"
```

## 使用方法

### 触发方式

用户发送问题时，此Skill会自动：
1. 验证API Key
2. 提取用户问题
3. 转发到Server OpenClaw
4. 返回答案

### 必填条件提醒

当用户请求生成志愿单、推荐志愿时：
- 如果缺少**省份**，自动追问
- 如果缺少**选科**（物理类/历史类），自动追问
- 如果缺少**分数**或**位次**，自动追问

### 首次使用提示

如果用户未配置API Key：
> ⚠️ 请先配置您的API Key才能使用蜻蜓志愿数据查询服务。如需API Key请联系管理员。

## API Key申请

联系管理员获取API Key。

## 测试

```bash
# 测试连接
python3 qingting_query_skill.py --api-key tk_xxx

# 测试问题
python3 qingting_query_skill.py -q "辽宁物理类520分能上什么学校" --api-key tk_xxx
```
